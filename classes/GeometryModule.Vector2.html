Static tripleProduct

  • tripleProduct(a, b, c): Vector2

  • The triple product of Vector2s is defined as:

    a x (b x c)

    However, this method performs the following triple product:

    (a x b) x c

    this can be simplified to:

    -a * (b &middot; c) + b * (a &middot; c)

    or:

    b * (a &middot; c) - a * (b &middot; c)

    Parameters

    • a: Vector2

      The a Vector2 in the above equation

    • b: Vector2

      The b Vector2 in the above equation

    • c: Vector2

      The c Vector2 in the above equation

    Returns Vector2

    The result of the triple product

    • Defined in geometry/Vector2.ts:190
